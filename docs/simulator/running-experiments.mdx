# Running Experiments

These guided experiments walk you through the key discoveries of Stommel's 1961 work and modern extensions. Each takes 10–30 minutes.

> **Important reminder.** This simulator implements the **original Stommel relaxation model**, in which the bifurcation parameter is the **temperature difference ΔT = T₁ − T₂** (not freshwater flux F as in some simplified textbook treatments). Throughout these experiments you will *push the system across tipping points by adjusting T₂* (the polar temperature). Larger T₂ ⇒ smaller ΔT ⇒ weaker thermal drive ⇒ closer to collapse.

---

## Experiment 1: Finding the Tipping Point

**Objective**: Discover the temperature-difference threshold (ΔT_crit) where AMOC collapses.

**Time**: ~15 minutes

**Difficulty**: Beginner

### Steps

1. **Load baseline state**:
   - Click preset: **Normal AMOC** (T₂ = 5 °C, ΔT = 20 °C)
   - Click **Run Simulation**
   - Note q_sv ≈ 15 Sv on the AMOC bar chart (strong circulation)

2. **Incrementally warm the polar box** by dragging T₂ upward:
   - Drag T₂ to **8 °C** (ΔT = 17) → Run; q drops slightly; still strong
   - Drag to **10 °C** (ΔT = 15) → Run; q ≈ moderate
   - Drag to **12 °C** (ΔT = 13) → Run; q noticeably weak
   - Drag to **14 °C** (ΔT = 11) → Run; marginal
   - Drag to **16 °C** (ΔT = 9) → Run; **q crashes to near zero** (collapsed!)

3. **Identify the collapse threshold**:
   - The T₂ value at which q suddenly falls is roughly **T₂ ≈ 14–16 °C** (i.e. ΔT_crit ≈ 9–11 °C with default Stommel coefficients)

4. **Fine-tune**:
   - Zoom in: try T₂ = 13.5, 14, 14.5, 15.0, 15.5
   - Find the exact T₂ where collapse occurs

5. **Verify analytically**:
   - Click **Compute Bifurcation Diagram** — the red diamond on the upper branch marks the precise ΔT_crit
   - Compare to your hand-found value

### What You're Seeing

As T₂ increases (ΔT decreases):
- **Below the threshold (large ΔT)**: Upper branch is stable; q decreases gradually
- **At ΔT_crit**: Upper and lower branches collide (saddle-node bifurcation)
- **Above the threshold (small ΔT)**: Upper branch no longer exists; system jumps to the lower (collapsed) branch

### Discussion

Why doesn't the AMOC weaken gradually all the way to zero? Why a sudden cliff?

**Nonlinear feedback.** As ΔT shrinks, the haline (salinity) feedback eventually becomes strong enough that the circulation cannot maintain itself. At the bifurcation point, the upper-branch attractor disappears — there is nowhere to be except the lower branch.

## Experiment 2: Demonstrating Hysteresis

**Objective**: Show that recovery from collapse requires a *larger* ΔT than the value at which collapse occurred — the smoking gun of irreversibility.

**Time**: ~20 minutes

**Difficulty**: Beginner–Intermediate

### Steps

1. **Collapse the AMOC**:
   - Load preset **Collapsed** (T₂ = 20 °C, ΔT = 5 °C)
   - Run simulation
   - Note q_sv ≈ 0 Sv (collapsed state)

2. **Try to recover by reducing T₂ back toward the collapse threshold**:
   - Drag T₂ slider down to **15 °C** (close to the threshold from Experiment 1)
   - Run simulation
   - **Observation**: q_sv stays low. AMOC does **not** recover yet!
   - The system "remembers" its collapsed initial condition

3. **Continue cooling the pole**:
   - Drag to **10 °C** (ΔT = 15) → Run; still weak/collapsed
   - Drag to **5 °C** (ΔT = 20) → Run; sometimes recovers, sometimes not (depends on initial salinity drift)
   - Drag to **0 °C** (ΔT = 25) → Run; **q snaps back up to ~15 Sv** (recovered!)

4. **Identify the recovery threshold**:
   - The T₂ value at which the lower branch dies and you snap back is the **backward (recovery) bifurcation**
   - With default Stommel parameters this often sits near T₂ ≈ 0–5 °C (ΔT ≈ 20–25 °C)

5. **Calculate the hysteresis window in ΔT**:
   - Hysteresis width = ΔT_recovery − ΔT_collapse
   - For example: 23 − 10 = **13 °C of ΔT**
   - That's the range of ΔT in which two stable states coexist

### What You're Seeing

The S-curve has three branches:
- **Upper branch** (strong AMOC): exists for ΔT > ΔT_collapse
- **Lower branch** (collapsed/haline): exists for ΔT < ΔT_recovery
- **Overlap (bistable region)**: ΔT_collapse < ΔT < ΔT_recovery — both stable branches coexist

Inside the overlap, your fate depends on **history** (initial conditions), not just current parameters. That is hysteresis.

### Discussion

**Why does hysteresis matter for climate?**

If the real AMOC is bistable and we currently sit inside (or near) the bistable window, then:
- Stopping greenhouse forcing won't automatically restore a collapsed AMOC
- Cooling the pole back to today's temperatures isn't enough — recovery requires going *colder than where we collapsed*
- Recovery is therefore harder and more costly than initially thought

## Experiment 3: Effect of Stochastic Forcing

**Objective**: Demonstrate how random noise can induce transitions even without changing mean parameters.

**Time**: ~15 minutes

**Difficulty**: Intermediate

### Steps

1. **Start with a weak but stable state**:
   - Load preset **Weakened** (T₂ = 12 °C, ΔT = 13 °C)
   - This sits near (or just inside) the bistable window
   - Open the **Stochastic Forcing** section and set noise_amplitude = **0**
   - Run 3 times — all trajectories converge to the same weak-but-stable state

2. **Add modest noise**:
   - Set noise_amplitude = **1 × 10⁻⁵**
   - Run simulation 3 times
   - Trajectories fluctuate around the attractor but don't escape

3. **Increase noise**:
   - Set noise_amplitude = **3 × 10⁻⁵**
   - Run 5 times
   - On at least one of these runs you should see q in the time-series plot suddenly fall off a cliff and stay collapsed for the rest of the run
   - This is **noise-induced tipping**

4. **Verify reproducibility**:
   - Run 5 more times — you should see a mix of "stays up" and "tips" outcomes
   - Each run uses a different random seed

5. **Quantify**:
   - Out of 10 runs, count how many tipped
   - Increase noise to **5 × 10⁻⁵** and count again — the rate goes up

### What You're Seeing

**Critical slowing down**: Near a bifurcation the dominant eigenvalue → 0, so recovery from perturbations is slow. Noise can exploit this weakness and push the system over the unstable manifold separating the two attractors.

**Stochastic shift of the effective tipping point**: With noise, transitions begin happening at parameters where the deterministic system would still be safely on the upper branch.

### Discussion

Earth's climate is full of noise: weather, eddies, volcanic eruptions, decadal modes (NAO, AMO), unresolved sub-grid processes. Even if the deterministic AMOC is far from ΔT_crit, noise can trigger early collapse. This motivates **precautionary** climate policy.

## Experiment 4: Exploring the Bifurcation Diagram

**Objective**: Visualise and understand the S-curve across the full range of polar temperatures.

**Time**: ~10 minutes

**Difficulty**: Beginner

### Steps

1. **Start from any configuration** (e.g. Normal AMOC preset)

2. **Click "Compute Bifurcation Diagram"** (the blue button)
   - The backend runs ~400 mini-simulations (200 forward, 200 backward) sweeping T₂
   - Takes ~10–30 seconds depending on hardware

3. **Examine the bifurcation diagram** (bottom-right panel):
   - **Blue curve**: Forward sweep (T₂ low → high, i.e. ΔT high → low) — upper-branch equilibria
   - **Orange curve**: Backward sweep (T₂ high → low, starting from a fresh polar condition) — lower-branch equilibria
   - **Grey shaded band**: Bistable region (hysteresis window)
   - **Red diamonds**: Tipping points
   - **Orange X**: Your current ΔT from the T₁ and T₂ sliders

4. **Locate the hysteresis window**:
   - Identify the collapse-side diamond (where the upper branch ends as ΔT shrinks)
   - Identify the recovery-side diamond (where the lower branch ends as ΔT grows)
   - The grey band between them is the bistable region

5. **Trace different paths**:
   - Move T₂ left and right → watch the orange X move along the diagram
   - When you land inside the grey band, you're in a parameter set where two stable states can coexist

### What You're Seeing

This is the **S-curve** (fold bifurcation diagram) Stommel drew in 1961. It captures the entire dynamics in one plot:
- Every stable equilibrium as a function of ΔT
- The location of tipping points
- The width of the hysteresis window

### Variations

1. **Change parameters and recompute**:
   - Reduce α to ~1 × 10⁻⁴ and recompute
   - Observation: the S-curve shifts; collapse threshold moves
   - Interpretation: weaker thermal expansion makes the system more fragile

2. **Increase β**:
   - Bump β to 1.5 × 10⁻³, recompute
   - Stronger haline contraction → larger bistable window, easier to tip

## Experiment 5: Recreating the Younger Dryas

**Objective**: Use the model to understand how an abrupt paleoclimate cooling could be triggered by perturbing ΔT.

**Time**: ~20 minutes

**Difficulty**: Intermediate–Advanced

### Scenario

The **Younger Dryas** (~12,900–11,700 yr BP) was a sudden return to glacial-like cold conditions after ~1,000 years of warming. The standard hypothesis: a meltwater outburst (Lake Agassiz) freshened the North Atlantic and collapsed the AMOC.

In the relaxation model we don't have an explicit freshwater flux, but we can mimic the effect: a freshwater pulse simultaneously **lowers polar salinity** and (via reduced sea-ice insulation) **raises polar temperature**. Both effects shrink the density contrast.

### Steps

1. **Set up an initial warm, stable state**:
   - Load **Normal AMOC** (T₂ = 5 °C)
   - Run — confirm q ≈ 15 Sv

2. **Apply a "freshwater pulse" by perturbing the pole**:
   - In the Temperature & Salinity section, set **T₂ = 18 °C** (mimicking warming + ice-insulation loss)
   - Set **S₂ᵢⁿⁱᵗ = 36.5** (mimicking a fresh-water dump that freshened the surface; here we use the *initial* salinity, not the eq value, because the pulse is transient)
   - Run simulation
   - **Observation**: q collapses

3. **Observe consequences**:
   - q falls from ~15 Sv toward 0 (or negative) over a few hundred years
   - Polar salinity drifts toward the equilibrium target via the relaxation term
   - Equatorial salinity adjusts as well

4. **Try slow recovery** (the Younger Dryas lasted ~1,200 years):
   - With T₂ still at 18 °C, set t_max = **10000** years
   - Run again
   - **Observation**: AMOC remains collapsed for thousands of years

5. **Force recovery by cooling the pole back down** (simulating the ice sheet finishing its meltwater discharge and the regional climate re-cooling):
   - Set T₂ back to 0 °C, t_max back to 3000
   - Run
   - **Observation**: AMOC suddenly re-strengthens once ΔT crosses the recovery diamond

6. **Interpret timings**:
   - Collapse: a few hundred years
   - Stuck-collapsed phase: as long as ΔT remains small (here, indefinite)
   - Recovery: a few hundred years once ΔT crosses the recovery threshold
   - Qualitatively matches the Younger Dryas event

### What You're Seeing

The Younger Dryas likely involved:
1. Massive freshwater discharge → density gradient destroyed → AMOC collapses
2. Hysteresis traps the system in the cold state
3. Freshwater input wanes as the ice sheet stabilises
4. ΔT eventually grows enough to re-cross the recovery threshold
5. AMOC abruptly re-starts → rapid rewarming

Stommel's two-box model, simple as it is, captures this qualitative sequence.

### Discussion

**Why use such a simple model for paleoclimate?**

Reality is complex: 3D ice sheets, coupled atmosphere, biogeochemistry, etc. But the **essential instability** (bistability + hysteresis) appears in even the simplest model. That suggests the nonlinearity is **fundamental**, not an artefact of oversimplification.

## Challenge Experiments

Once you've completed the five main experiments, try these:

### Challenge A: Bifurcation Sensitivity

- Compute the bifurcation diagram with default parameters; record the two diamond positions
- Change one parameter at a time (α, β, k, λ, S₂ᵉᵍ)
- Recompute and see how much the tipping points shift
- Which parameter has the biggest effect?

### Challenge B: Noise-Induced Early-Warning Signals

- Set T₂ = 13 (just above the collapse threshold)
- With noise_amplitude = 3 × 10⁻⁵, run 10 times
- Compare the time series — do you see *increased variance or autocorrelation* (red noise) in q before tipping events? These are the canonical Early Warning Signals from the literature.

### Challenge C: Multi-Timescale Dynamics

- Set t_max = 20000 years
- Set T₂ inside the bistable window (e.g. 10 °C)
- With noise_amplitude = 2 × 10⁻⁵, run a single simulation
- Do you see multiple transitions between states? What's the typical residence time in each state?

### Challenge D: Parameter Uncertainty

- Pick a preset; add realistic noise
- Vary several parameters by ±10% from their defaults
- Do the conclusions (location of tipping points, hysteresis width) change significantly?

## Exporting Results

There is currently no Export button in the UI. To save data, hit the API directly:

```python
import requests, json

params = {
    "T_1": 25, "T_2": 5,
    "S_1_eq": 36, "S_2_eq": 34,
    "S_1_init": 36, "S_2_init": 34,
    "alpha": 2.0e-4, "beta": 1.0e-3,
    "k": 3.0e-9, "lam": 3.0e-11,
    "t_max": 3000, "dt": 1.0,
    "noise_amplitude": 0.0
}

resp = requests.post('http://localhost:8000/api/simulate', json=params)
data = resp.json()

with open('simulation_export.json', 'w') as f:
    json.dump(data, f, indent=2)

# Quick plot
import matplotlib.pyplot as plt
plt.plot(data['time'], data['q_sv'])
plt.xlabel('Time (years)'); plt.ylabel('Circulation (Sv)')
plt.show()
```
