# Getting Started

## System Requirements

Before installing, ensure you have:

- **Python ≥ 3.11** ([download](https://www.python.org/downloads/))
- **Node.js ≥ 18** ([download](https://nodejs.org/))
- **`uv` package manager** (install via `pip install uv` or [see docs](https://docs.astral.sh/uv/getting-started/installation/))
- **Git** (to clone the repository)

Verify installation:

```bash
python --version      # Should be 3.11+
node --version        # Should be 18+
npm --version         # Should be 9+
uv --version          # Should be 0.1.0+
```

---

## Clone and Navigate

```bash
git clone https://github.com/your-org/ocean-sciences.git
cd ocean-sciences
```

(Replace with your actual repository URL.)

---

## Backend Setup

The backend is a **FastAPI** server that runs simulations. Set it up in a separate terminal:

### 1. Create a Virtual Environment

```bash
cd backend
uv sync
```

This command:
- Creates a `.venv` virtual environment
- Installs all dependencies from `pyproject.toml`
- Takes ~30 seconds on first run

### 2. Start the Server

```bash
uv run uvicorn api:app --reload
```

**Expected output**:
```
INFO:     Uvicorn running on http://0.0.0.0:8000
INFO:     Application startup complete
```

The `--reload` flag enables hot-reloading during development; remove it for production.

### 3. Verify the Server

In a third terminal, test connectivity:

```bash
curl http://localhost:8000/api/health
```

**Expected response**:
```json
{"status": "ok"}
```

## Frontend Setup

The frontend is a **React + Vite** web app. Set it up in a separate terminal:

### 1. Install Dependencies

```bash
cd frontend
npm install
```

This installs ~163 packages listed in `package.json`. Takes ~1–2 minutes on first run.

### 2. Start the Development Server

```bash
npm run dev
```

**Expected output**:
```
VITE v5.4.21 ready in 205ms

➜  Local:   http://localhost:5173/
➜  press h + enter to show help
```

The Vite dev server automatically reloads when you edit files.

### 3. Open in Browser

Navigate to **http://localhost:5173** in your web browser.

### Click "Run Simulation"

1. Leave all parameters at defaults
2. Click the **"Run Simulation"** button
3. Watch the visualizations populate:
   - **Top-left**: Two-box ocean diagram with animated arrows
   - **Top-right**: Comparison of model's q_sv vs. observed RAPID mean
   - **Middle**: Time series of q, S_e, S_p vs. time (years)
   - **Bottom-left**: Phase space portrait (ΔS vs. q)
   - **Bottom-right**: [empty until you click "Compute Bifurcation"]

The simulation took ~1–2 seconds. The system quickly equilibrated to a steady state (note the flat time series near t=3000 years).

## Understanding the Layout

### Left Panel: Controls

- **Parameters sliders**: Adjust T_e, T_p, S_e0, S_p0, α, β, k, F, t_max, dt, noise_amplitude
- **Log scale toggle**: k has a logarithmic slider (exponential scale)
- **Preset buttons**: Load "Normal AMOC", "Weakened AMOC", "Collapsed AMOC", "Stommel 1961"
- **Run Simulation**: Execute the simulation with current parameters
- **Compute Bifurcation**: (advanced) Sweep F to generate bifurcation diagram
- **Export**: Download current result as JSON

### Right Panel: Visualizations (6 panels)

See [Visualizations](/simulator/visualizations) for detailed explanation of each.

    "S_e0": 36,
    "S_p0": 34,

## Troubleshooting

### Issue: "Port 8000 is already in use"

**Solution**: Either kill the process on port 8000 or use a different port:

```bash
# Find and kill process on port 8000 (macOS/Linux)
lsof -ti:8000 | xargs kill -9

# Or use a different port
uv run uvicorn api:app --reload --port 8001
```

### Issue: "uv: command not found"

**Solution**: Install uv globally:

```bash
pip install uv
uv --version
```

Or use the installation script:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

### Issue: "npm install fails" or takes forever

**Solution**: Clear npm cache and try again:

```bash
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
```

### Issue: "Python version mismatch" (need 3.11+)

**Solution**: Check your Python version and install a newer one if needed:

```bash
python --version
# If < 3.11, install Python 3.11 or 3.12
```

On macOS with Homebrew:
```bash
brew install python@3.12
brew link python@3.12
```

### Issue: Browser shows blank page / "Cannot GET /"

**Solution**: Make sure the frontend dev server is running and on port 5173:

```bash
cd frontend
npm run dev
# Verify output shows "Local: http://localhost:5173/"
```

Then clear browser cache (Ctrl+Shift+R or Cmd+Shift+R) and reload.

### Issue: Simulations show "Failed to fetch" errors

**Solution**: Ensure backend is running on port 8000:

```bash
# Terminal 1
cd backend
uv run uvicorn api:app --reload

# Check connectivity
curl http://localhost:8000/health
```

The frontend expects the backend at `http://localhost:8000`. If using a different port, update the Vite config or backend server URL.
